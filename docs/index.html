📚 新書新刊情報 RSS フィード

このサイトでは、openBD APIから取得した新書の新刊情報をRSSフィードとして配信しています。

RSS フィードを購読する

特徴

  • 毎日午前4時に自動更新
  • 新書の新刊情報のみを厳選して配信
  • 書名、著者、出版社、シリーズ名などの詳細情報を含む
  • 著者略歴も含めた充実した内容

フィードの登録方法

以下のURLをお使いのRSSリーダーに登録してください：

https://YOUR_USERNAME.github.io/shinsho/index.xml

対応しているRSSリーダー

  • Feedly
  • Inoreader
  • NewsBlur
  • The Old Reader
  • その他、標準的なRSSリーダー
